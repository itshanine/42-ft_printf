<h2>
  🛠️ Usage
</h2>

``make`` to compile <b>without</b> bonus functions.

``make clean`` to delete all object files.

``make fclean`` to delete object files and <b>libftprintf.a</b>.

``make re`` to delete everything and rebuild <b>ft_printf</b>.
